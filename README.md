# 📊 Commercial Sales Dashboard | Power BI

## 📌 Overview
This project presents an interactive Commercial Sales Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales performance, profitability, and key business metrics through dynamic visualizations and KPI tracking.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor revenue and profit trends.
- Identify top-performing products and regions.
- Enable data-driven business decisions.
- Create an interactive and user-friendly dashboard.

---

## 🛠 Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📈 Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Quantity Sold
- Number of Orders
- Profit Margin
- Sales by Region
- Sales by Category
- Monthly Sales Trends

---

## 📊 Dashboard Features
✔ Interactive Filters and Slicers

✔ Revenue and Profit Analysis

✔ Regional Sales Performance

✔ Category-wise Sales Analysis

✔ Trend Analysis

✔ Dynamic Visualizations

✔ Business Insights for Decision Making
